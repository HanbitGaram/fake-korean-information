# 책임의 부인
- 이 코드를 사용해서 책임을 지게 되어야 하는 경우, 제작자는 책임을 지지 아니합니다.
- 이름 및 데이터는 현지화 데이터이며, 무작위로 생성되나 개인의 정보와 유사할 수 있습니다.
